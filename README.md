# autoWorker

### Fully Automated Cloudflare Worker Deployment CLI

Deploy and manage Cloudflare Workers with a single command.
No manual setup, no complicated configuration, and no repetitive dashboard work.

---

## Overview

**AutoWorker** is a fully automated CLI tool designed to simplify Cloudflare Worker deployment.

Unlike traditional deployment workflows that require multiple manual steps, AutoWorker handles the entire process automatically:

- OAuth authentication
- Worker creation
- Configuration generation
- Deployment

Simply download the latest binary from the Releases page and run it.

No Node.js installation required.
No additional dependencies required.

---

## Features

- Fully automated Cloudflare Worker deployment
- Interactive CLI interface
- OAuth-based authentication
- Automatic Worker configuration
- Automatic deployment workflow
- No manual dashboard configuration
- No Wrangler setup required

---

## Installation

Download the latest binary from GitHub Releases

Then run:

### Linux

```bash
chmod +x autoworker
./autoworker
```

### Windows

```powershell
autoworker.exe
```

### macOS

```bash
chmod +x autoworker
./autoworker
```

---

## Nova Proxy Integration

AutoWorker can automatically deploy and configure projects based on:

[IRNova/Nova-Proxy](https://github.com/IRNova/Nova-Proxy)

Instead of manually creating Workers and configuring everything yourself, AutoWorker automates the deployment workflow and significantly reduces setup time.
